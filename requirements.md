# Discord投稿プレビューWebアプリ 要件定義

## 概要

Discordのアナウンスや通常メッセージを、本家UIに近い見た目でプレビューできるWebアプリ。
Discordタイムスタンプ記法の変換機能も備える。

---

## 技術方針

- **外部依存なし**：ライブラリ・フレームワーク不使用（バニラHTML/CSS/JS）
- **単一ファイル構成**：`index.html` 1ファイルで完結（配布・共有が簡単）
- **ブラウザのみで動作**：サーバー不要、ローカルで開くだけで使える
- **GitHub Pages対応**：静的ファイルのみで構成し、そのままデプロイ可能

---

## デプロイ構成

```
リポジトリ/
├── index.html        # アプリ本体（単一ファイル）
├── README.md         # 使い方説明
└── .github/
    └── workflows/
        └── deploy.yml  # GitHub Actions（gh-pagesブランチへ自動デプロイ）
```

- `main` ブランチへの push で GitHub Actions が自動デプロイ
- GitHub Pages の公開ソースは `gh-pages` ブランチ（または `main` の `/root`）
- カスタムドメイン設定も可能（`CNAME` ファイル追加で対応）

## 機能要件

### 1. テキスト入力エリア

- Discordのメッセージ入力欄に似たテキストエリア
- リアルタイムプレビュー（入力のたびに即時反映）
- 投稿者名・アイコン色（または画像URL）を設定できる
- チャンネル名の表示切替（アナウンスチャンネル風ヘッダー）

### 2. Discordマークダウン対応

以下の記法をパースしてプレビューに反映する：

| 記法 | 表示 |
|---|---|
| `**テキスト**` | **太字** |
| `*テキスト*` / `_テキスト_` | *斜体* |
| `__テキスト__` | 下線 |
| `~~テキスト~~` | 取り消し線 |
| `\`コード\`` | インラインコード |
| ` ```言語\nコード\n``` ` | コードブロック |
| `> テキスト` | 引用（blockquote） |
| `>>> テキスト` | 複数行引用 |
| `||テキスト||` | スポイラー（クリックで表示） |
| `-# テキスト` | 小テキスト（subtext） |
| `# テキスト` | 大見出し（H1） |
| `## テキスト` | 中見出し（H2） |
| `### テキスト` | 小見出し（H3） |
| `- テキスト` / `* テキスト` | 箇条書き |
| `1. テキスト` | 番号付きリスト |
| `@ユーザー名` | メンション風ハイライト |
| `#チャンネル名` | チャンネルリンク風ハイライト |
| `:絵文字名:` | 絵文字（Unicode対応のみ） |

### 3. Discordタイムスタンプ変換

#### 入力形式

- Unixタイムスタンプを入力すると `<t:TIMESTAMP:FORMAT>` 記法を生成
- または日時ピッカーから選択して生成

#### フォーマット対応

| 記法 | 表示例 |
|---|---|
| `<t:TIMESTAMP>` / `<t:TIMESTAMP:f>` | 2025年1月1日 12:00 |
| `<t:TIMESTAMP:F>` | 2025年1月1日水曜日 12:00:00 |
| `<t:TIMESTAMP:d>` | 2025/01/01 |
| `<t:TIMESTAMP:D>` | 2025年1月1日 |
| `<t:TIMESTAMP:t>` | 12:00 |
| `<t:TIMESTAMP:T>` | 12:00:00 |
| `<t:TIMESTAMP:R>` | 3時間前 / 2日後（相対表示） |

#### タイムスタンプ変換ツール（サイドパネル or タブ）

- 日時ピッカー → 全フォーマットのDiscord記法を一覧表示
- ワンクリックでコピー
- 現在時刻をワンクリックで入力

### 4. プレビュー表示

- Discordダークテーマ（#313338 系）で再現
- ユーザーアイコン（色付き丸またはカスタム画像URL）
- ユーザー名（カラー付き）・タイムスタンプ
- アナウンスチャンネル風バッジ（任意）
- コードブロックはシンタックスハイライトなし（外部依存回避）またはシンプルな色分け

### 5. UI / UX

- 左：入力エリア、右：プレビュー の2カラムレイアウト（PC向け）
- スマホでは上下に切替タブ
- 「クリップボードにコピー」ボタン（テキストをコピー）
- プレビューのスクリーンショット風コピー（将来対応）
- ダークモード固定（Discordに合わせる）

---

## 非機能要件

- **ファイルサイズ**：index.html 単体で 200KB 以内
- **動作環境**：Chrome / Firefox / Edge / Safari 最新版
- **オフライン動作**：インターネット接続不要
- **アクセシビリティ**：キーボード操作でタイムスタンプ生成可能
