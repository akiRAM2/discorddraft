# DiscordDraft

Discordの投稿（複数メッセージ、マークダウン、OGPカード、タイムスタンプ）を、ブラウザ上でリアルタイムに下書きプレビューするための単一HTMLツールです。

公開URL: `https://akiram2.github.io/discorddraft/`

## 機能

- **Discordマークダウン対応** — 太字・斜体・下線・取り消し線・コードブロック・引用・スポイラー・見出し・リストなど
- **複数メッセージ** — 複数ユーザーの会話を並べてプレビュー
- **OGPプレビュー** — URLを貼るとOGPカードを表示（`<URL>` で抑制）
- **入力欄の自動拡張** — 入力に合わせて縦に伸長（一定以上はスクロール）
- **カラムリサイズ** — 左右カラム幅をドラッグで調整（ブラウザに保存）
- **Discordタイムスタンプ変換** — 日時を選ぶと全7フォーマット(`<t:...>`)を生成
- **Free / Nitro 文字数制限** — 2000字 / 4000字の切り替え
- **スポイラー** — クリックで表示

## 使い方

1. `https://akiram2.github.io/discorddraft/` をブラウザで開く（またはローカルで `index.html` を直接開く）
2. 左パネルでメッセージを入力 → 右パネルにリアルタイムプレビュー
3. **Timestamp** タブでDiscordタイムスタンプを生成

## セキュリティ/プライバシー

- このツール自体にサーバーはなく、入力したメッセージ本文を「ツールのサーバー」に送信・保存する仕組みはありません（ブラウザ内で完結します）。
- ただし **OGPプレビューをONにしてURLを含める** と、カード生成のためにそのURLを外部へ取得しに行きます（CORS回避のため公開プロキシを利用）。
  - 外部送信を完全に避けたい場合は、OGPプレビューをOFFにして使ってください。

## OGPについて（重要）

- このツールのOGPカードは「Discordの実際の埋め込み」と一致することを保証しません。
- 取得はブラウザから外部サイトへアクセスするため、ネットワークが必要です（オフラインだとOGPは出ません）。
- 取得にはCORS回避のための公開プロキシを利用します（環境や相手サイト側の制限で失敗することがあります）。
- X/Twitter はそのままだとOGP取得が不安定なため、取得時に `x.com -> fixupx.com`、`twitter.com -> fxtwitter.com` に置換してOGPを取得します。

## マークダウン記法

| 記法 | 表示 |
|---|---|
| `**太字**` | **太字** |
| `*斜体*` | *斜体* |
| `__下線__` | 下線 |
| `~~取り消し~~` | ~~取り消し線~~ |
| `\|\|スポイラー\|\|` | クリックで表示 |
| `` `コード` `` | インラインコード |
| ` ```js\nコード\n``` ` | コードブロック（Prism.jsハイライト） |
| `> 引用` | 引用 |
| `>>> 複数行引用` | 複数行引用 |
| `# 見出し1` | H1 |
| `-# 小テキスト` | subtext |
| `@ユーザー名` | メンション |
| `#チャンネル` | チャンネルメンション |
| `<t:1234567890:f>` | Discordタイムスタンプ |
| `<URL>` | OGPなしリンク |
| `[表示](https://example.com)` | ハイパーリンク（本家Discordの通常メッセージとは挙動が異なります） |

## デプロイ

`main` ブランチへの push で GitHub Actions が自動的に GitHub Pages へデプロイします。

Settings → Pages → Source: **GitHub Actions** に設定してください。
